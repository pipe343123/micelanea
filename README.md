# Base de Datos de Micelánea

Este proyecto presenta la base de datos micelanea, diseñada para gestionar de manera ordenada los productos de una miscelánea. Su objetivo es ofrecer una estructura simple, clara y fácil de consultar.

 # Estructura General

La base de datos está compuesta por dos tablas principales que permiten organizar los productos según su categoría:

### 1. Tabla: tipo_productos

Aquí se registran las categorías disponibles para clasificar los productos.
Incluye dos tipos:

* Oficina

* Escolar

### 2. Tabla: productos

En esta tabla se almacenan todos los artículos de la miscelánea, junto con el tipo al que pertenecen mediante el campo id_tipo.

El inventario está compuesto por:

* 10 productos de oficina

* 10 productos escolares

# Propósito del Proyecto

La base de datos micelanea tiene como finalidad mantener una gestión organizada del inventario, permitiendo clasificar los productos de manera eficiente y facilitando la consulta y administración interna de la miscelánea.
